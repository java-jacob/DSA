Recursion
----------
A programming technique where by which A method calls itself. Each call solves a small part of big problem.
Conceptual framework for solving many problems.

1. Calculate triangular numbers.
2. Generate Anagram.
3. Find Factorials.
4. Recursive binary search.
5. Solve Tower of Hanoi.
6. Merge sort.
